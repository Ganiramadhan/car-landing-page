# GaniPedia Car Catalog

GaniPedia Car Catalog is a simple car catalog application built using ReactJS. This application allows users to view a list of available cars along with their details such as name, price, and description. Users can also use the credit simulation feature to estimate credit costs.

## Features

- View a list of cars along with their details.
- Credit simulation for estimating credit costs.

## Getting Started

1. Clone this repository to your local system.
2. Open a terminal and navigate to the cloned repository directory.
3. Run the command `npm install` to install all required dependencies.
4. Once finished, run the command `npm start` to start the application.
5. The application will run on `http://localhost:3000` by default.
